**Weather App**

A simple and interactive weather application that allows users to check the current weather conditions for any location. The app fetches real-time weather data using a weather API and displays the current temperature, weather conditions, and more.

**Features**

Search weather by city name or location.

View current temperature and weather conditions (e.g., sunny, cloudy, rainy).

Responsive design for desktop and mobile devices.

Displays weather information with real-time updates.


**Technologies Used**

HTML5 - Structure and layout of the application.

CSS3- Styling the UI.

JavaScript - Client-side scripting.

OpenWeatherMap API - Fetching weather data.


**Usage**

Open the app in your browser.

Enter a city name in the search bar.

Click the search button to retrieve and display the current weather data.

View the temperature and weather conditions.

The app will update automatically based on the latest weather data.
